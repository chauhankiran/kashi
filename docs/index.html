Kashi

The black & white style framework v0.0.4

Kashi is a small black & white style framework created for small websites, email templates and blogs where you need style but without features burden. It's lightweight and hackable in nature. You can easily hack it by overriding the existing set of variable defined within css.

Getting Started

Getting started with Kashi is easy. You just need to install it's package via npm. 

$ npm install kashi

Or you can use CDN to include within your existing/new project.

<link rel="stylesheet" href="https://unpkg.com/kashi@latest">

That's it! You are good to go. Mentioned methods are not working for you? File an issue.

Button

Kashi includes one predefined button style applies on input, button and a. Just drop a one class - btn and you are good to go. In case needed, you can use btn-lg make it large!

<input type="text" class="btn" value="input[button]">

<button type="submit" class="btn">Submit</button>

<a href="#"class="btn">Link</a>

<button type="submit" class="btn" disabled>Disable Submit</button>

<a href="#" class="btn btn-lg">Large Link</a>

Utilities

Kashi has many utilities components that help you with single and common repetitive tasks such as center aligning, capitalizing text or floating an image on given side. Here is the complete list of these utilities classes that bundle with Kashi.

  • .text-left - left align the text.
  • .text-center - center align the text.
  • .text-right - right align the text.
  • .text-justify - justify the text.
  • .text-lowercase - transfer text to lowercase
  • .text-uppercase - transfer text to uppercase
  • .text-capitalize - transfer text to capitalize
  • .text-brand - colored text with brand color
  • .text-muted - colored text with gray color
  • .block - modified the display properly to block.
  • .inline-block - modified the display properly to inline-block.
  • .inline - modified the display properly to inline.
  • .none - modified the display properly to none or hide the applied element
  • .rectangle - removed any applied border-radius.
  • .rounded - apply a little border-radius to given element.
  • .circle - circle the element by applying 50% border-radius.
  • .left - floats an element to the left
  • .right - floats an element to the right
  • .text-big - increase the size of text, a little bit.
  • .text-small - decrease the size of text, a little bit.
  • .text-normal - normalize the given text by its size, style and weight.
  • .text-bold - bold the text by its weight.
  • .text-italic - italic the text by its style.
  • .text-thin - light the text by its weight.
  • .invisible - invisible the text by hiding its visual-ness.

Do you think one or more helper class(es) is/are missing? Please file an issue.
