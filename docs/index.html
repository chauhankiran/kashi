Kashi

A small hackable style framework v0.0.1

Kashi is a small style framework created for small website, email template and blog where you need style but without features burden. It's lightweight and hackable in nature. You can easily hack it by overriding the existing set of variable defined within css.

Getting Started

Getting started with Kashi is easy. You just need to install it's package via npm. 

$ npm install kashi

Or you can use CDN to include within your existing/new project.

<link rel="stylesheet" href="https://unpkg.com/kashi@latest">
