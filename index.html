Installation

Download and installation instructions.

There are 2 ways you can start using Kashi in your websites and services.

  1. Install with package manager (recommended)

    We recommend installing Kashi via package manager - node package manager (npm), or yarn.

    $ npm i kashi

  2. Install by using compiled files

    If you are not using Kashi via npm, you can install/CDN by using our compiled files.

    <link rel="stylesheet" href="https://unpkg.com/kashi@latest">
