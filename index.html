Kashi, The black & white style framework

Kashi is a style framework created with two colors in mind and simplicity as aim.
It includes base CSS and HTML for typography, forms, buttons, tables, grids, navigation, and more.

Nerd alert: Kashi is built with Stylus as hack to provides easy customization and to remove hurdle of set-up.

Hotlink the CSS

For the quickest and easiest start, just copy this snippet into your webpage.

<link rel="stylesheet" href="https://unpkg.com/kashi@latest">
npm package

Install Kashi in your Node.js powered apps with the npm.

$ npm i kashi

About Kashi Brief history, browser support, and more

Kashi was partially created as part of learning the inners of all popular styles frameworks. While creating one component at a time, it becomes so big that it stands for its own!

Browser support

Kashi is tested and supported in major modern browsers like Chrome, Brave, and Firefox.

  • Latest Google Chrome
  • Latest Firefox
  • Latest Brave

What's included

Kashi comes complete with compiled CSS, uncompiled, and example templates.

  • All original .styl files.

  • Examples pages for each component
  • Three example pages with different layouts
  • Three themes

Layouts Basic templates to create webpages

Fixed layout

The default and simple 60rem-wide, centered layout for just about any website or page provided by a single <div.container>.

<body>
    <div class="container">
        ...
    </div>
</body>

Fluid layout

An alternative, flexible fluid page structure with min- and max-widths and a left-hand sidebar. Great for apps and docs.

<body>
    <div class="container-fluid">
        <div class="sidebar">
            ...
        </div>
        <div class="content">
            ...
        </div>
    </div>
</body>

Typography Headings, paragraphs, lists, and other inline type elements

Headings

A standard typographic hierarchy for structuring your webpages.

The entire typographic grid is based on two Less variables in our variables.less file: @basefont and @baseline. The first is the base font-size used throughout and the second is the base line-height.

We use those variables, and some math, to create the margins, paddings, and line-heights of all our type and more.
