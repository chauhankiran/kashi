# Kashi

> The black & white style framework

## Getting Started

You can easily install Kashi via `npm`.

```bash
$ npm i kashi
```

Or you can link it via CDN.

```html
<link rel="stylesheet" href="https://unpkg.com/kashi@latest">
```

That's it! You're good to go.
